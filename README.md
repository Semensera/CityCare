🏙️ CityCare — Система моніторингу міських проблем
CityCare — це бекенд-додаток на базі Spring Boot, розроблений для фіксації та автоматичного управління міськими запитами (мітками). Проект використовує MongoDB для гнучкого зберігання даних та реалізує механізм автоматичного видалення застарілих записів.

🚀 Основні можливості
Зберігання міток у NoSQL: Використання MongoDB для швидкої роботи з даними.

Механізм TTL (Time To Live): Мітки автоматично видаляються з бази даних після закінчення терміну дії, який вказує користувач.

REST API: Зручний інтерфейс для створення та отримання списку актуальних проблем міста.

Архітектура DTO: Використання об'єктів передачі даних для чистоти коду та безпеки.

🛠️ Технологічний стек
Java 17

Spring Boot 3.x

Spring Data MongoDB

Maven (управління залежностями)

MongoDB (база даних)
